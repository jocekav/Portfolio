Campus Involvements

Featured skills: Python, HTML, JavaScript, SQL, Heroku, Project Management

HeartBeats is a dating app that matches users based on music-tastes. Based on our users' listening histories and music preferences, we develop matches and joint playlists for an immediate icebreaker.
I worked on the project in a team of two, and I focused on the architecture and implementation the app. I wrote the backend powering the application in Python and the front-end in HTML/CSS and JavaScript. The application runs with Python Flask and MySql, and it is hosted on Heroku.

Individual Contribution

The application allows the user to create an account, log in, and view other users. Each user's account is tied to their Spotify account. We collect a user's Spotify listening history, playlists, and liked songs using the Spotify API.
The user can scroll through other profiles and views their match score, the other user's top songs, and location. When I user choses a match, they are given a joint playlist in Spotify which they can play right on the HeartBeats site via the Spotify WebSDK.
I chose to use Flask for the backend to utilize Python for the matching algorithm, SQL for user data, and standard web programming for the view. I deployed the app on Heroku for the demo period and user testing. I designed the GUI initially in Figma, then built the functional version using the JUCE visual library. The GUI was connected using JUCE's value state tree to control the preset, efffect, and model states.
