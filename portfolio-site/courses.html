Featured Courses

Georgia Tech

Audio Software Engineering

This course taught the fundamentals of programming for audio including system architecture, building real-time systems, testing frameworks, and SCRUM development. In this course, I implemented a comb filter, ring buffer delay line, vibrato, dynamic time warping, fast convolution, and spectrogram in C++. I also created the Dynamic Guitar Amp plugin using C++ and the JUCE library in a group of four.

Artificial Intelligence

This course detailed AI algorithms inlcuding various search algorithms, minimax, bayes nets, decision trees, gaussian mixture models, and hidden markov models. I developed the algorithms and creating testing suites for each in Python.

Media Device Architecture

In this course, I learned how to interface between hardware and software in C by building games for the Gameboy Advance. Concepts covered the memory allocation, code structure, compilation, graphics, and audio. For the final, I created a game top scrolling game similar to Doodle Jump.

Audio Content Analysis

I studied music information retrieval in this course. I implemented pitch tracking, feature extraction, key detection, and genre classifcation algorithms in Python. With a group, I developed an automatic classifier for Beijing Opera which included implementing tempo detection using the discrete wavelet transform.

Digital Signal Processing

In this course, we focused on DSP for audio. I learned fundamental DSP algorithms, and I implemented correlation, convolution, and Fourier analysis in Python. I also created a command line synth with a partner featuring additive synthesis, wavetable synthesis, reverb, LFO, flanger, ring modulation, and high and low pass filters.

Video Game Design

During this course, I developed a video game with a team of four while learning about game development. In our game, the player navigates as a robot through multiple levels, unlocking new mechanics as they complete puzzles. I focused on character movement, sound effects, an anti-gravity mechanic, and a throwing mechanic.
